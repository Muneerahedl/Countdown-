# CountDown | كم باقي 

## Challenge Statement
Help any person to build a personal reminder counter.

## Solution Concept
An App that helps / provides a way for any person who wants to make a personalized countdown by create the countdown and view them.
.

## Technologies & Tools
- localization
- VoiceOver
- Core Data
- SwiftUI
